# Zscore calculator
This was done as a part of the interview process for wiserfunding.

## How to run
- Set up the python virtual environment
  ```
  python -m venv venv
  source venv/bin/activate
  ```
- Install dependencies from requirements.txt
  ```
  pip install -r requirements.txt
  ```
- Run uvicorn server
  ```
  uvicorn main:app
  ```
